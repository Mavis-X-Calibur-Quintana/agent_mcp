Terminal操作规范

1. 严禁使用`rm`、`rm -rf`、`rm -r`、`rm -f`等删除命令！！！

2. 执行脚本前，请先使用 close_terminal_if_open 关闭所有终端，再使用 open_new_terminal 打开一个新的终端。

3. 使用 run_script_in_exist_terminal 在终端内执行脚本语句，执行完成后使用 get_terminal_full_text 查看执行结果（会包含当前终端中的所有文本）。

4. 如果发现需要交互的场景，使用 send_terminal_keyboard_key 向终端发送控制命令，常用的控制命令如下：
4.1 键盘向上：up
4.2 键盘向下：down
4.3 键盘向左：left
4.4 键盘向右：right
4.5 回车键：return
4.6 空格键：space

5. 工具 send_terminal_keyboard_key 调用时需要传入一个按键数组，示例如下：
```python
# 向上点击键盘
send_terminal_keyboard_key(["down"])

# 输入Ctrl+C终止异常流程
send_terminal_keyboard_key(["key code 8 using {{control down}}"])

# 先输入向下，再输入回车
send_terminal_keyboard_key(["down", "return"])
```

此时如果要选中“React”，可以通过向Terminal输入两次向下按键后，再按回车键实现，方法如下：
```python
send_terminal_keyboard_key(["key code 125", "key code 125", "return"])
```
也可以分三次输入：
```python
send_terminal_keyboard_key(["key code 125"])
send_terminal_keyboard_key(["key code 125"])
send_terminal_keyboard_key(["return"])
```

6. 出于安全考虑以下命令被禁止使用：
```bash
- 文件系统破坏命令（rm, rmdir, del）
- 磁盘/文件系统命令（format, mkfs, dd）
- 权限/所有权命令（chmod, chown）
- 权限提升命令（sudo, su）
- 代码执行命令（exec, eval）
- 系统通信命令（write, wall）
- 系统控制命令（shutdown, reboot, init）
```
